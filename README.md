# Experiment-5-Toggle-Button

Aim

To create an Android application that uses a Toggle Button control to switch between two states and display the current status using a Toast message.

Algorithm
1.Launch Android Studio and create a new project.
2.Open the layout file activity_main.xml.
3.Use a RelativeLayout as the parent layout.
4.Insert a ToggleButton widget into the layout.
5.Set the ON and OFF text for the ToggleButton.
6.Open MainActivity.java.
7.Declare a ToggleButton variable.
8.Connect the ToggleButton using findViewById().
9.Attach a listener using setOnCheckedChangeListener().
10.Detect the state change of the ToggleButton.
11.If the button is turned ON, display a Toast message "Toggle is ON".
12.If the button is turned OFF, display a Toast message "Toggle is OFF".
13.Execute the application on an emulator/device.
14.Observe the output when the toggle state is changed.
